To use code snippets through the snippet’s shortcut
    In the Visual Studio IDE, open the file that you intend to change.
    In the Code Editor, put the cursor where you want to insert the code snippet.
    Type the shortcut for the code snippet that you want to add to your code.
    OR –
    Press CTRL+SPACE to view the completion list and then select the snippet shortcut from the list.
    Press TAB, TAB to invoke the code snippet.
